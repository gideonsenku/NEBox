# Changelog

## 1.0.0

### 新功能和修复

- 支持从剪贴板粘贴或文件导入备份数据，告别大文本卡顿问题
- 支持将备份数据导出为 JSON 文件，通过系统分享面板保存或发送
- 在系统"文件"等 app 中可直接将 JSON 文件分享到 Relay 导入
- 导入过程中显示居中 HUD 加载提示，实时展示导入进度
- 优化初始启动时网络权限获取流程，提升用户体验

### 改进

- 启动速度优化，减少界面卡顿和不必要的重绘
- 滚动浏览体验更加流畅
- 自动识别当前代理工具（Loon / Surge / Shadowrocket / Quantumult X）

### 修复

- 修复首次启动时 iOS 尚未授予网络权限导致的连接失败
- 修复部分页面无法侧滑返回的问题
- 修复订阅页面导航和上下文菜单的交互异常
- 兼容 iOS 15 及以上版本
