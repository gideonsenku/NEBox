# Changelog

## 1.0.0

### 新功能

- 全新原生界面，管理 BoxJS 脚本和订阅更直观
- 支持 JSON 文件导入导出备份数据，也可通过系统分享面板发送
- 在系统"文件"等 app 中可直接将 JSON 文件分享到 Relay 导入
- 内置脚本编辑器，随时查看和运行脚本
- 新增数据查看器，浏览和搜索存储数据
- 新增应用日志查看与导出，方便排查问题
- 支持自定义 BoxJS 偏好设置，包括 Surge HTTP-API 配置
- 适配 iOS 26 Liquid Glass 设计风格

### 改进

- 全面优化性能，滚动更流畅、加载更快
- 主屏幕风格的抖动编辑模式，轻松排列收藏应用
- 订阅卡片支持拖拽排序
- 首次启动时更智能地处理网络权限延迟
- 自动识别当前代理工具（Loon / Surge / Shadowrocket / Quantumult X）

### 修复

- 修复部分页面无法侧滑返回的问题
- 修复订阅页面导航和时间刷新异常
- 兼容 iOS 15 及以上版本
